# Third-party components and reference assets

## Bundled libraries

- Bootstrap 5.3.8 — MIT License — https://getbootstrap.com/
- Bootstrap Icons 1.13.1 — MIT License — https://icons.getbootstrap.com/
- GSAP 3.13.0 en ScrollTrigger — GSAP Standard “No Charge” License — https://gsap.com/standard-license/

De geminificeerde distributiebestanden zijn lokaal opgenomen onder `assets/vendor/`, zodat Bootstrap en de scrollanimatie niet afhankelijk zijn van een CDN. Controleer voor productiegebruik altijd de actuele GSAP-licentievoorwaarden.

## Webfonts

De demo laadt Montserrat, Poppins en Roboto via Google Fonts. Deze families zijn open-source, maar de request blijft extern. Host de fonts lokaal als je een volledig zelfstandige of streng privacygerichte productieversie nodig hebt.

## Reference photography

De HTML verwijst voor visuele demonstratiedoeleinden naar publiek bereikbare foto-URL's onder:

`https://bootstrapmade.com/content/demo/Stratify/assets/img/`

Deze beeldbestanden zijn niet opgenomen in dit project en vallen niet onder de MIT-licentie van de zelfstandig geschreven theme-code. Hotlink deze afbeeldingen niet in een productiewebsite. Vervang ze door eigen fotografie of door beelden waarvoor je de benodigde gebruiks- en distributierechten hebt.

## Visual reference

De openbare Stratify-demopagina diende als visuele en structurele referentie. Dit pakket bevat geen originele commerciële templatebroncode. Controleer afzonderlijk de licentievoorwaarden van de originele aanbieder als je originele assets of originele templatebestanden wilt gebruiken.
